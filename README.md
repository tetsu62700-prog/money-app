# 家計簿アプリ

## 概要
Pythonで作成したターミナル操作型の家計簿アプリです。
収入・支出の記録、編集、削除、集計、JSON保存に対応しています。

## 使用技術
- Python
- JSON

## 主な機能
- 記録の追加
- 記録一覧の表示
- 記録の編集
- 記録の削除
- 支出合計の表示
- 収入合計の表示
- カテゴリ別支出の表示
- 月別集計
- 月別カテゴリ支出の表示
- 日付順の自動並び替え
- JSONによる自動保存
- 起動時のデータ読み込み
- 入力チェック

## ファイル構成
money_app/
├── main.py
├── records.py
├── storage.py
└── money_records.json

## 実行方法
以下のコマンドで実行します。

python main.py

## 工夫した点
- 機能ごとに関数を分け、保守しやすいコードを意識しました。
- 保存処理を storage.py に分離しました。
- 記録操作を records.py に分離しました。
- 金額や日付の入力ミスでアプリが落ちないように、入力チェックを実装しました。
- 記録を追加・編集・削除したタイミングで自動保存されるようにしました。

## 学んだこと
-リストと辞書を使ったデータ管理
-JSONファイルへの祖損と読み込み
-関数分割による保守性向上
-try/exceptによる入力エラー対応
-Git/GitHubを使ったバージョン管理

## 苦労した点
